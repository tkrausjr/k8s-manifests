In preparation for deploying one or more TKG Extensions, download TKG Extensions Bundle from VMware.

The TKG Extensions are packaged as a separate bundle that you download from VMware.

Procedure

  1. On the system that you use as your Kubernetes client machine and run kubectl comands, go to https://www.vmware.com/go/get-tkg.
  2. Log in with your My VMware credentials.
  3. Under Product Downloads, click Go to Downloads.
  4. Scroll to the entry for VMware Tanzu Kubernetes Grid Extensions Manifest X.X.X.
  5. Select the latest version or the version you are targeting and click Download Now.
  6. Download the extensions to the same location where you run kubectl commands.
  7. Use either the tar command or the extraction tool of your choice to unpack the bundle of YAML manifest files for the TKG extensions. 
    tar -xzf tkg-extensions-manifests-vX.X.X-vmware.1.tar.gz
  8. Verify the TKG Extensions download.
    Navigate to the following path and verify the /extensions directory. This directory is the home path for the TKG Extensions. 
    /Downloads/tkg-extensions-manifests-vX.X.X-vmware.1.tar/tkg-extensions-manifests-vX.X.X-vmware.1/tkg-extensions-vX.X.X+vmware.1/extensions
    Important: In the documentation for installing and updating TKG Extensions, the home path is referred to as /tkg-extensions/extensions. In this directory are the various subdirectories for each of the extensions as shown below.
